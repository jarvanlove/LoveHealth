# 配料表+营养食谱APP开发计划

## 项目概述
- 项目名称：配料表+营养食谱APP
- 开发周期：30天
- 技术栈：Node.js(18.0+) + Express + MySQL(5.7) + Minio + JWT + Redis(3.0+) + Vue3 + Vite + Pinia + Vue Router + Vant UI + Axios + Capacitor
